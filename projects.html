BukkitCommons

BukkitCommons is a library of utilities designed to ease programming using the Bukkit API.

BukkitCommons is licensed under the Apache License 2.0, is open sourced, and available for download both under the "Downloads" in the navbar, and on Bukkit Dev

BukkitCommons has development builds for every commit pushed to GitHub that passes the compiler, thanks to TravisCI, the build script, and the github-maven-plugin
